# Temporary assets

Place **synthetic or public-domain CSV files** here for local testing and demos.

## Rules

- **Do not commit private, confidential, or production datasets.**
- Real datasets are ignored by `.gitignore` (`*.csv` except explicitly allowed samples).
- Only files in this directory can be selected via the Assets UI.
- The backend validates paths and rejects directory traversal.

## Included samples (added during implementation)

- `pest_control_sample.csv` — demo data for business analytics / pest workflow (synthetic).
